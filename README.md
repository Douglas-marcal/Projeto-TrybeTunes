# Projeto TrybeTunes

Projeto em React desenvolvido no módulo de Front-end durante o curso de desenvolvimento web na Trybe.

### Foi possível praticar:

- Consumir e manipular dados vindos de API;
- Estados em React;
- Ciclos de vida;
- Rotas com React Router;
